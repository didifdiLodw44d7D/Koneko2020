# ３切りこねこ

### 概要

三目並べのゲームです。
文字を並べて高得点を狙います。

#### ルール

置く場所：
	どこにおいてもかまいません。

得点：
	縦でも、横でも、斜めでも。
	「こ」「ね」「こ」が成立したら５点です。
	「ね」「こ」が成立したら２点です。

勝敗：
	点数が多い方が勝ちです。

※2020-02-15現在
	先手番の対CPU線のみ可能です。
